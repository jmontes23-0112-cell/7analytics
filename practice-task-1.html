Task Description

Build a responsive navigation bar with smooth scrolling functionality. The navigation should include links to different sections of the page and feature a glassmorphism effect with backdrop blur.

Key Requirements:

  • Fixed navigation with backdrop blur effect
  • Smooth scrolling between sections
  • Responsive design for mobile and desktop
  • Gradient text effects for branding
  • Hover animations and transitions

Source Code

<!DOCTYPE html>
<html>
<head>
    <title>Responsive Navigation</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gradient-to-br from-purple-900 to-indigo-900">
    
    <nav class="fixed top-0 w-full z-50 bg-white/10 backdrop-blur-md border-b border-white/20">
        <div class="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8">
            <div class="flex justify-between items-center py-4">
                <div class="text-2xl font-bold bg-gradient-to-r from-purple-400 to-pink-400 bg-clip-text text-transparent">
                    Jason C. Montes
                </div>
                <div class="hidden md:flex space-x-8">
                    <a href="#home" class="hover:text-purple-300 transition-colors">Home</a>
                    <a href="#about" class="hover:text-purple-300 transition-colors">About</a>
                    <a href="#projects" class="hover:text-purple-300 transition-colors">Projects</a>
                    <a href="#contact" class="hover:text-purple-300 transition-colors">Contact</a>
                </div>
            </div>
        </div>
    </nav>

    <script>
        // Smooth scrolling navigation
        document.querySelectorAll('a[href^="#"]').forEach(anchor => {
            anchor.addEventListener('click', function (e) {
                e.preventDefault();
                const target = document.querySelector(this.getAttribute('href'));
                if (target) {
                    target.scrollIntoView({
                        behavior: 'smooth',
                        block: 'start'
                    });
                }
            });
        });
    </script>

</body>
</html>
